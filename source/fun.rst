Fun module
==============


me
-----------------
    Shows user's profile.

Arguments
~~~~~~~~~
  user: Union[:class:`Mention`, :class:`Name`, :class:`ID`]

Returns
~~~~~~~~~
  An image with user's profile.

Aliases
~~~~~~~~~
  - profile

Example
~~~~~~~~~
  - !me
  - !me Naneynonn
  - !profile @Naneynonn#0101
  - !profile 496569904527441921
